## 代码规范

### Python 代码规范

- 遵循 PEP 8 风格指南
- 使用 4 个空格缩进
- 最大行长度：88 字符（Black 格式化器）
- 使用类型注解

**示例**：

```python
def function_name(param1: str, param2: int) -> bool:
    """函数文档字符串。
    
    Args:
        param1: 参数1说明
        param2: 参数2说明
        
    Returns:
        返回值说明
    """
    pass
```

### 提交信息规范

格式：`[类型]: [简短描述]`

类型：
- `feat`: 新功能
- `fix`: 错误修复
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 代码重构
- `test`: 测试相关
- `chore`: 构建/工具相关
