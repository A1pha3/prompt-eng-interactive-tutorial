## 使用建议

### 术语一致性

在开发文档中使用术语时，请遵循以下原则：

1. **首次出现提供英文**：专业术语首次出现时，提供英文原文
   ```markdown
   虚拟环境（Virtual Environment）用于隔离项目依赖。
   ```

2. **保持一致性**：在整个文档中使用相同的译名
   ```markdown
   # 正确
   使用虚拟环境管理依赖...
   激活虚拟环境后...
   
   # 错误
   使用虚拟环境管理依赖...
   激活 Virtual Environment 后...
   ```

3. **专有名词保留原文**：工具名称、产品名称等保留英文
   ```markdown
   使用 Black 格式化代码
   运行 pytest 执行测试
   ```

4. **代码中使用英文**：代码、命令、文件名使用英文
   ```markdown
   运行 `pip install black` 安装 Black
   ```
