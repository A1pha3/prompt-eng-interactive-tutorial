## 术语使用规范

### 1. 专有名词保留原文

以下术语建议保留英文原文，不进行翻译：
- Claude
- Bedrock
- Boto3
- Jupyter Notebook
- Markdown
- Python
- API（可在首次出现时标注中文）

### 2. 中英文混排规范

- 中英文之间添加空格：`使用 Claude 模型`
- 代码、命令、文件名使用等宽字体：`` `requirements.txt` ``
- 专业术语首次出现时提供英文原文：`提示工程（Prompt Engineering）`

### 3. 标点符号使用

- 中文内容使用中文标点：`这是一个示例。`
- 代码和英文内容使用英文标点：`This is an example.`
- 列表项末尾不加标点（除非是完整句子）

### 4. 术语一致性

在整个文档体系中，同一概念应使用统一的中文术语。参考本术语表确保一致性。
